<CaseCard label="Overview" index="01">

2D action game built in 32 hours for Global Game Jam 2025. You play as a bubble. Firing parts of yourself is how you attack, which is also how you lose health. Size, speed, and survival all tied to the same resource.

</CaseCard>

<CaseCard label="My Role" index="02">

Game designer, systems designer, 2D artist, and animator.

- Core size / speed / health trade-off.
- Self-firing weapon mechanic that costs health to use.
- Enemy archetypes with distinct behaviour patterns.
- Bubble drops, spawn patterns, difficulty scaling.
- All art and UI for the game.
- Design documentation so the team could align under jam pressure.

</CaseCard>

<CaseCard label="Outcome" index="03">

Shipped by the jam deadline. The team kept extending it past the initial build.
